Describe the difference between C# and .NET Framework in 2-3 sentences:

C# is a programming language, intended to be a simple, modern, general-purpose, object-oriented programming language. Dot net (.NET) is a framework library, including a large class library known as Framework Class Library (FCL) which provides language interoperability. Programs written for .NET Framework execute in a software environment, known as Common Language Runtime (CLR). FCL and CLR together constitute .NET Framework.
